04. Sistemas de numeración

Sistemas numéricos

Los sistemas numéricos son formas de representar cantidades usando diferentes bases. Cada sistema tiene un conjunto de símbolos (dígitos) y una base que indica cuántos símbolos se usan. 

Tenemos 3 bases que vamos a utilizar: 

  • Decimal 

  • Hexagesimal  

  • Binario o Diatico 

1. Sistema Decimal (Base 10) 

  • Es el sistema que usamos todos los días. 

  • Usa 10 dígitos: del 0 al 9. 

  • Cada posición representa una potencia de 10 (unidad, decena, centena…). 

Ejemplo: 

345 = 3×10² + 4×10¹ + 5×10⁰ 
 

 2. Sistema Binario (Base 2) 

  • Se utiliza en electrónica, computadoras y automatización. 

  • Usa 2 dígitos: 0 y 1 (representan apagado y encendido). 

  • Cada posición representa una potencia de 2. 

Ejemplo: 

1011₂ = 1×2³ + 0×2² + 1×2¹ + 1×2⁰ = 11₁₀ 

3. Sistema Hexadecimal (Base 16) 

  • Se usa en informática, electrónica digital y programación. 

  • Usa 16 símbolos: 

  • Del 0 al 9 (igual que el sistema decimal). 

  • De la A a la F, donde: 
    A = 10 
    B = 11 
    C = 12 
    D = 13 
    E = 14 
    F = 15 

  • Cada dígito hexadecimal representa 4 bits. 

Usos comunes: 

  • Direcciones de memoria 

  • Colores en HTML y CSS (por ejemplo: #FF5733) 

  • Códigos en lenguaje máquina
